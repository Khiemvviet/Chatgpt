Project Introduction

This project combines web design excellence with rigorous AI evaluation to analyze ChatGPT's efficiency across multiple domains including History, Social Science, and Computer Security.

Research Approach

Our methodology employs a comprehensive tech stack:

  • MongoDB for structured data storage
  • Node.js & Express for server-side operations
  • ChatGPT API for model interaction
  • WebSocket for real-time communication
  • Chart.js for data visualization

Dataset Overview

We evaluate ChatGPT across three specialized domains with 50+ questions each:

  • History: Historical events, causes, and figures
  • Social Science: Sociological concepts and theories
  • Computer Security: Cybersecurity principles and practices
