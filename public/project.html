Methodology

We systematically evaluated ChatGPT's performance by:

  1. Populating MongoDB with domain-specific questions and expected answers
  2. Routing questions through Node.js server to ChatGPT API
  3. Measuring response accuracy and timing metrics
  4. Storing and analyzing results in MongoDB
  5. Visualizing insights through interactive charts
